# AgriTech AI: Smart Farming Dashboard

![Dashboard Preview](https://hassanahmed2ha.github.io/-AgriTech-AI-Hassan-Ahmed-Prototype-/#dashboard)

## Overview
**AgriTech AI** is a **smart farming platform** built by **Hassan Ahmed Hassan Zaki**, an 18-year-old researcher from Alexandria, Egypt.  
It helps farmers **save 20% water** using **IoT simulation + AI insights**.

> **Live Demo**: [Open Dashboard](https://hassanahmed2ha.github.io/-AgriTech-AI-Hassan-Ahmed-Prototype-/#dashboard)

---

## Features
- Real-time **soil moisture & yield charts**
- **AI Chat Advisor** (English/Arabic)
- **Dark & Light Mode**
- **Mobile-Friendly Dashboard**
- **Ambient farming sounds**

---

## Tech Stack
- HTML, CSS (Tailwind), JavaScript
- Chart.js, Font Awesome
- GitHub Pages
